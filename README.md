# Zakat Calculator

This   is a simple Android application to calculate Zakat on gold savings. It's built with Kotlin and follows modern Android development practices.

## Features

- Calculate Zakat for two types of gold savings: "Keep" and "Wear".
- Input the weight of gold in grams.
- Input the current value of gold per gram.
- Displays the total value of the gold.
- Displays the total Zakat payable.
- A share button to share the app with others.
- An "About" page with developer information.

## How to Use

1.  Enter the weight of your gold in grams.
2.  Select the type of gold: "Keep" or "Wear".
3.  Enter the current value of gold per gram.
4.  Click the "Calculate" button.
5.  The app will display the total value of your gold and the Zakat payable.

## Technologies Used

- Kotlin
- Android SDK
- Material Design Components
